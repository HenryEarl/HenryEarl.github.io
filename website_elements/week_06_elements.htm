Website Elements

This Webpage design proved to be the most challanging so far. I had initial intentions to make my horizontal list encapsalated with a border. I started by looking at other websites for help. I did find examples for utilizing the .horizontal selector from Knowlscodes. I felt that copying the design and tweaking it to fit my needs would be plagarism. All of my attempts to modify the background color for this single element failed. I tried for several hours while flying to North Carolina to visit friends and family. I ended up abandoning this idea in order to focus on the overall layout of the paragraphs. After adding webfonts to the stylesheet, this proved to be very useful. I was successfully able to move the text away from the centered border. In my opinion it is more pleasing to have structure in my page over stylish borders and matching colors. Both would make my page more visualy aeshetic, I felt that the latter issue to be more important.

Trusted Tools

The tool I used the most was the element inspector provided by my browser, FireFox. This inspection tool allowed me to change elements and move things around without having to constantly save and refresh pages. The inspection tool also allowed me to inspect other webpages for reference purposes. Seeing how others use style sheets to move elements around really played a huge role in my success for completing this project.

Another useful utility at my disposal was the pdf version of our text. Being able to quickly search for keywords to find references to help guide my desing were crucial.

Final Thoughts

When creating my Website Elements page for this week, I decided to use the same style and layout sheets created for the Choir homepage. I felt this would save me a lot of valuable time. This is also useful in learning how to design future websites that will encorporate the same style sheets. A fear that comes to mind is changing this page to utilize my home page format. But understanding style sheets should make this obstacle bearable if needed.

I would also recommend linking the Rubric for Assignments to your websites if availble. This was also very useful when adding the finishing touches to my webiste.
